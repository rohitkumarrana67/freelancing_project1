CampusNet - Student Management System

CampusNet SMS is an extensive automation solution for Educational Campus; touching every detail of a student's overall development inside a Campus, analyzing activities from various angles such academic, behavioral and extra-curricular excellence. CampusNet at the back end takes care of all transaction involving Finance, Inventory and Human Resource Management.

Built on, .Net technology, CampusNet works on n-tier architecture and the application can run on Cloud or as a Client Server system that can be live updated into the Web with fool proof data encryption. Plug in is provided to integrate Embedded machines or instruments such as Bio-Metric Finger Print Devices or Smart Card/ Proximity Card based devices.

CampusNet combines these three essentials to give the added advantage to the Students, Parents and Management:

⇨ The Campus Automation Solution

⇨ Web Interface

⇨ Plug in for Embedded Devices
